Simple Sockets

Welcome to simple-sockets. This is my first time working with Socket.io, and has quickly become one of my favorite projects. This is my MVP (minimum-viable-product) of a chat app.

How to:

Set a username, then join or create a room. Passwords are saved hashed in a server side room object

Usernames are unique across the app, and are freed for someone else when you disconnect.

Connections and messages are not persistent. If you refresh or loose connection, you will need to set a username again.

Rooms are currently persistent until server reset.
